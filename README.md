# search-nicovideo-by-sinatra

[ニコニココンテンツ検索API](http://search.nicovideo.jp/docs/api/search.html) を使ったサンプル
使ってるライブラリ https://github.com/iwag/search-nicovideo-rb

## local

```bash
$ bundle install
$ bundle exec ruby app.rb
```

http://localhost:4567

## heroku

```bash
$ heroku create
$ git push heroku master
$ heroku open
```
